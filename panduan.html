Tentang Aplikasi

Aplikasi Database RTLH (Rumah Tidak Layak Huni) Kabupaten Kediri adalah platform digital yang memudahkan akses data rumah tidak layak huni di seluruh wilayah Kabupaten Kediri. Aplikasi ini menyediakan akses cepat ke data spreadsheet Google Sheets yang tersimpan secara terorganisir berdasarkan Kecamatan dan Desa.

Tip: Aplikasi ini dirancang untuk memudahkan pencarian dan akses data RTLH tanpa perlu mengingat link atau mencari file satu per satu.

Cara Menggunakan Aplikasi

Mencari Data Berdasarkan Kecamatan

1

Buka Halaman Utama

Pada halaman utama, Anda akan melihat daftar semua Kecamatan yang tersedia di Kabupaten Kediri.

2

Gunakan Fitur Pencarian (Opsional)

Ketik nama Kecamatan di kolom pencarian untuk memfilter daftar dan menemukan Kecamatan yang Anda cari dengan cepat.

3

Pilih Kecamatan

Klik tombol "Lihat Desa" pada kartu Kecamatan yang ingin Anda akses. Setiap kartu menampilkan jumlah desa yang tersedia.

4

Pilih Desa

Setelah memilih Kecamatan, Anda akan melihat daftar Desa. Gunakan pencarian jika diperlukan, lalu klik "Akses Data RTLH".

5

Akses Google Sheets

Link akan membuka Google Sheets di tab baru. Pastikan Anda sudah login ke akun Google yang memiliki akses.

6

Konfirmasi

Jika sudah selesai melakukan verifikasi/pendataan lakukan verifikasi kepada Admin. Admin akan memeriksa, menyelaraskan dengan E-RTLH dan mengirimkan rekap PDF untuk disetujui

Pastikan KK dan NIK dalam TEXT atau ditambahkan ' (petik) di depan angka contoh : '3506.... .

Tutorial Mengecek DTSEN via Website PKP

Langkah 1: Akses Website PKP

Buka browser Anda dan akses link berikut: https://my.pkp.go.id/cekbantuan

Langkah 2: Masukkan Data NIK

Anda memiliki dua pilihan untuk memasukkan data:

  • Input Manual: Masukkan NIK satu per satu pada kolom yang tersedia
  • Upload Excel: Siapkan template Excel berisi kumpulan NIK yang akan dicek. Download template di: https://my.pkp.go.id/template_data_cek.xlsx

Langkah 3: Submit Data

Setelah data NIK dimasukkan atau file Excel di-upload, klik tombol "Submit" untuk memproses pengecekan status penerima bantuan.

Langkah 4: Catat Status DTSEN

Sistem akan menampilkan hasil pengecekan untuk setiap NIK. Masukkan Status hasil yang muncul pada kolom DTSEN sesuai dengan NIK yang dicari di data Anda.

Catatan Penting: Pastikan NIK yang diinput sudah benar dan sesuai dengan data KTP untuk mendapatkan hasil yang akurat.

Tips & Trik

  • Gunakan Keyboard: Tekan Ctrl + F atau Cmd + F untuk pencarian cepat di halaman browser.
  • Akses Offline: Setelah dibuka sekali, beberapa elemen aplikasi akan di-cache sehingga loading lebih cepat pada kunjungan berikutnya.
  • Login Google: Pastikan Anda sudah login ke akun Google yang memiliki akses ke spreadsheet sebelum mengklik tombol "Akses Data RTLH".
Perhatian: Jika Anda tidak memiliki akses ke Google Sheets, hubungi admin untuk mendapatkan permission.

Pertanyaan yang Sering Diajukan (FAQ)

Bagaimana jika saya tidak bisa membuka Google Sheets?
Pastikan Anda sudah login ke akun Google dan memiliki permission untuk mengakses spreadsheet tersebut. Jika masih bermasalah, hubungi admin untuk mendapatkan akses.
Apakah saya bisa mengedit data di Google Sheets?
Hak edit tergantung pada permission yang diberikan oleh pemilik spreadsheet. Jika Anda memerlukan akses edit, hubungi admin untuk pengaturan permission.
Data untuk Desa tertentu tidak muncul, kenapa?
Kemungkinan data untuk desa tersebut belum diinput ke dalam database atau sedang dalam proses pembaruan. Hubungi admin untuk informasi lebih lanjut.
Apakah aplikasi ini memerlukan koneksi internet?
Ya, aplikasi ini memerlukan koneksi internet untuk mengakses Google Sheets. Namun, interface aplikasi dapat di-cache oleh browser untuk loading yang lebih cepat.
Siapa yang dapat saya hubungi untuk bantuan teknis?
Hubungi admin melalui tombol WhatsApp yang tersedia di halaman ini atau di halaman utama untuk bantuan dan konfirmasi pengerjaan.

Troubleshooting

Masalah Umum dan Solusinya

Problem: Halaman tidak loading
  • Cek koneksi internet Anda
  • Refresh halaman dengan Ctrl + R atau Cmd + R
  • Clear cache browser dan coba lagi
  • Coba gunakan browser yang berbeda
Problem: Link Google Sheets tidak bisa dibuka
  • Pastikan Anda sudah login ke Google Account
  • Cek apakah Anda memiliki permission untuk mengakses file
  • Hubungi admin untuk request akses
Problem: Pencarian tidak bekerja
  • Pastikan menggunakan browser modern (Chrome, Firefox, Safari, Edge)
  • Enable JavaScript di browser Anda
  • Refresh halaman
